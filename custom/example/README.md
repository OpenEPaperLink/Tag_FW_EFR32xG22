This "tag" consists of an [ePaper 7.4" BWRY display from Pervasive Displays Inc](https://www.mouser.com/ProductDetail/Pervasive-Displays/E2741QS0B3?qs=yc9RBI4tIALK9i8Hvi25SQ%3D%3D&srsltid=AfmBOoquM_jjZ_qT6GhV49Rq-cz-vvyWZV4k6qfZQ9VZe7gTsiiiLS2G)
housed in an [IKEA RODALM Picture Frame](https://github.com/OpenEPaperLink/OpenEPaperLink/wiki/Picture-Frame-for-7.4%22-Price-tags) with a
Solum controller salvaged from a 4.2" Tag powered by a CR123A lithium battery.

See the OEPL [wiki](https://github.com/OpenEPaperLink/OpenEPaperLink/wiki/DIY-Tags#diy-frankenstein-74-bwry-tag) for more details.
